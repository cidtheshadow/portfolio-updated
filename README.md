# Tanush Singla — Personal Portfolio

A performance-driven, aesthetically rich personal portfolio showcasing my engineering projects, systems programming work, and professional background. Designed with a focus on modern UI/UX, glassmorphism aesthetics, and clean component architecture.

## 🚀 Live Demo
*(Update with your deployed link, e.g., https://tanushsingla.com)*

## ✨ Key Features
- **Modern UI Architecture:** Built with Next.js App Router for optimal routing and performance.
- **Glassmorphism Design:** Custom CSS gradients, backdrop filters, and subtle glow effects to create a deep, premium dark mode aesthetic.
- **Scroll-Driven Animations:** Smooth entry animations powered by standard `IntersectionObserver`, avoiding heavy animation library overhead.
- **Recruiter-Focused Dashboard:** A dedicated Hero metrics section highlighting immediate key achievements and capabilities.
- **Responsive Layout:** Fully optimized for mobile, tablet, and desktop viewing.
- **Print-Friendly:** Custom CSS print styles ensure the portfolio renders as a clean, black-and-white resume document when printed directly from the browser.

## 🛠️ Technology Stack
- **Framework:** Next.js 15 (React)
- **Styling:** Tailwind CSS + Custom CSS Utilities
- **Icons:** Lucide React & Devicon
- **Typography:** Inter & Poppins (via Google Fonts)

## 💻 Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/cidtheshadow/portfolio-updated.git
   cd portfolio-updated
   ```

2. **Install dependencies**
   ```bash
   pnpm install
   # or npm install / yarn install
   ```

3. **Run the development server**
   ```bash
   pnpm run dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000) to see the application running.

## 📁 Project Structure
- `/app` - Next.js App Router layout, page structure, and global CSS.
- `/components` - Modular React components (Hero, About, Projects, Certifications, Contact, Header, Footer).
- `/public` - Static assets, profile images, and resume files.
